# 🏏 NextInningsIPL

NextInningsIPL is an intelligent voice-controlled IPL cricket assistant built using Python.

It provides live scores, player statistics, match schedules, points tables and predicts the next innings score using Machine Learning.

---

# 🚀 Features

✔ Voice Controlled Assistant  
✔ Live Cricket Score  
✔ IPL Match Schedule  
✔ IPL Points Table  
✔ Player Statistics  
✔ Player Comparison  
✔ Machine Learning Score Prediction  

---

# 🧠 Technologies Used

Python  
SpeechRecognition  
Scikit-Learn  
BeautifulSoup  
Requests  
NumPy  
Pandas  

---

# 📊 Machine Learning

This project uses **Linear Regression** to predict the next innings score based on previous IPL match data.

# ⚙ Installation

Clone the repository:

git clone https://github.com/niyal0903/NextInningsIPL.git


Go to project folder:

Install requirements:
pip install -r requirements.txt

Run project:
python main.py
